# 사용자 액세스키 생성

IAM 사용자 자격 증명인 액세스 키(Access Key)와 API 인증 토큰 발급 실습입니다.

## 1. 사용자 액세스키 생성


1. 우측 상단 계정 프로필 > `자격 증명` > 비밀번호 확인
2. `IAM 액세스 키 생성` 클릭
     - 프로젝트 지정 : `(나의 계정이 속한 프로젝트의 이름 선택)`
     - 사용자 액세스 키 이름 : `kakaocloud-test`
     - 만료 기한 : `무제한`
3. `생성` 버튼 클릭
4. `IAM 액세스 키 ID` 복사 후 메모장 등에 붙여넣기
5. `보안 액세스 키` 복사 후 메모장 등에 붙여넣기
6. `확인` 버튼 클릭
